---
title: "Quickstart"
description: "Configure sua primeira integração em 3 minutos"
---

<Steps>
  <Step title="Gerar sua API Key">
    Acesse o painel da Integra em **Settings > API Keys** e clique em **Nova API Key**.

    <Warning>
      Copie a key gerada (`ik_live_...`) imediatamente — ela **só será exibida uma vez**.
    </Warning>
  </Step>

  <Step title="Autenticar requests">
    Envie o header `Authorization` em todas as chamadas:

    ```bash
    Authorization: Bearer ik_live_sua_key_aqui
    ```
  </Step>

  <Step title="Fazer sua primeira chamada">
    ```bash
    curl -H "Authorization: Bearer ik_live_xxx" \
      https://api.useintegra.com.br/api/v1/campaigns
    ```

    <Check>
      Se a resposta for `200 OK` com JSON, sua integração está funcionando!
    </Check>
  </Step>
</Steps>
